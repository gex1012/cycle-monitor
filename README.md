# 时间周期择时监测平台 (Time-Cycle Market Timing Monitor)

把三种经典**时间周期**方法（合成周期 / Hurst 嵌套 / 斐波那契时间周期）实现为可计算代码，
对一篮子全球标的做"时间到位"择时监测，并用**时间共振 (Time Confluence)** 抬高拐点确信度。

> ⚠️ 这是周期/技术择时**启发式研究工具**，只负责"时间窗口 + 大盘方向"，不做估值/基本面。
> 板块与宏观对冲 overlay（半导体、原油/煤炭等）不在模型内。不构成投资建议。

---

## 一、快速开始 (Windows)

```bat
:: 1) 安装依赖（首次）
python -m pip install -r requirements.txt

:: 2) 启动平台 —— 双击 run_app.bat，或命令行：
```

> 侧栏每个参数、信号板每列也都有 ❓ 悬停提示。

## 三、模型原理（对应 `cycles.py`）

1. **合成周期 Composite Cycle** — 对数价格→线性去趋势→加窗 FFT 频谱分解→挑主导周期分量→
   最小二乘重构合成曲线并向前外推；合成曲线（纯周期分量）的**未来波谷=投影时间拐点**。
2. **周期嵌套 Hurst Nesting** — 带通振荡器探测历史波谷，估计微周期（默认搜索 40–150 交易日带，
   约 8–30 周）平均实测长度后向前投影；并叠加 2:1 嵌套名义周期（季 65d / 半年 130d / 年 260d）。
3. **斐波那契时间周期** — 从关键拐点(pivot)起按斐波那契交易日 (13,21,34,55,89,144,233,377,610...)
   向前数，投影下一时间拐点。pivot 默认自动探测近 ~2 年最显著摆动极值，可在 `config.py` 手动锁定。

**时间共振**：把三法投影的拐点按 ±容差 聚类，落在同一窗口的方法越多，分值越高
（🟢高 ≥2.5 / 🟡中 ≥1.5 / ⚪低）。这就是抬高确信度的标准逻辑。

## 四、首页怎么读

- **📅 近期时间窗口**：全标的共振点聚合的日历，按日期排序——哪天、哪些标的、几法共振、确信度。
- **📊 信号板**：每个标的的最新价/涨跌、趋势（vs长均线）、当前**周期相位**（上行▲/下行▼）、
  **下一波谷日期**、距波谷交易日数、最近共振窗口、以及一句**操作指引**
  （绿=偏多/建仓窗口，红=减仓，橙=下行观望，灰=中性）。

**标的详情**页给出价格 + 合成周期拟合/投影曲线、周期波谷/波峰标记、斐波时间线、
微周期投影线、共振阴影窗口，以及三法明细表与文字指引。

## 五、四个页面

1. **🏠 首页·信号板** — 全标的周期信号 + 📅 共振时间窗口日历 + **📡 最新宏观更新与各标的影响** +
   **📝 各标的·后续走势观点**（每个标的一段自然语言：合成周期反弹/见顶时间窗 + 当前第几轮微周期、约几周、何时结束）。
2. **🔬 标的详情** — 周期图（图上标注 **预计反弹/见顶窗口** 与 **微周期末端**）+ **📌 后续走势观点** +
   **📋 完整交易建议**（方向/时间窗/触发/入场/止损/目标/仓位）+ **🧠 分析总结** + 三法明细。
3. **🌍 宏观数据·季节性** — 重要宏观数据月度总结表（通胀/就业/消费/工业/货币/地产/增长/**耐用品订单及分项**/**PMI类制造业景气及分项**）
   + **季节性图（过去 N 年按日历月叠加：每年一条线 + 季节均线，今年加粗）** + 历史走势 + 对各标的影响。
   PMI 因 ISM 有版权未上 FRED，改用**费城/纽约联储制造业调查**（扩散指数，含新订单/出货/就业/支付价格等分项）作免费代理。
4. **📖 帮助·信号解读** — 信号/图例/参数/共振逐项讲解 + "7.10 怎么数出来的"案例。

## 六、交易建议 & 宏观说明（重要）

- **交易建议**由 `analysis.py` 自动生成：方向来自周期相位+趋势；时间窗来自合成/共振投影；
  关键价位（支撑/阻力/中枢/ATR）来自近端价格结构。**这些是模型推导的参考值，不是精确点位，更不是投资建议。**
- **宏观层**由 `macro.py` 提供（FRED 免密钥）：最新数据 vs 前值的方向，按规则映射到各资产桶，
  汇总成每个标的的"方向净评分"。这是**启发式宏观 overlay 评论**，与周期模型相互独立，同样不构成投资建议。
- 利率（10年美债收益率）为**收益率口径**：收益率↑=债价↓，建议文字里已按此语义标注。

## 七、文件结构

```
config.py    监测标的清单 + 默认参数（在此增删标的、改 pivot）
data.py      Yahoo Finance 抓取层（OHLC，带重试）
cycles.py    三套周期模型 + 时间共振 + 指引生成（核心数理）
analysis.py  完整交易建议 + 分析总结（周期结果 + 价格结构 + 宏观叠加）
macro.py     FRED 宏观数据 + 同比/环比变换 + 季节性 + 影响规则
app.py       Streamlit 仪表盘（首页 / 详情 / 宏观 / 帮助 四页）
run_app.bat  Windows 一键启动
```

## 六、常见问题

- **某标的"数据缺失"**：Yahoo 偶发限流/休市。点侧栏 🔄 刷新行情重试。
- **想加标的**：在 `config.py` 的 `INSTRUMENTS` 加一行（key/name/ticker/group/log/pivot）。
- **投影日期有几天漂移**：未来日期用工作日历外推，未剔除各市场节假日，长周期投影可能 ±数日，属正常。
- **缓存**：行情默认缓存 30 分钟；改参数会自动重算。
